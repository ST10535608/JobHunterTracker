# Job Hunt Tracker

An Android app that helps users monitor their weekly job search activity and stay motivated during their job hunt.

## About
Job Hunt Tracker allows users to log their daily job search data for each day of the week and receive a personalised summary at the end of the 7-day period.

## Features
- Log daily job search data for each day of the week
- Track applications sent, hours spent searching, and interviews attended
- Add notes for each day
- View a weekly summary including:
    - Total applications sent
    - Most productive day
    - Average hours spent searching per day
    - Personalised motivational feedback

## Screens
1. **Welcome Screen** — app introduction with Start and Exit buttons
2. **Daily Data Screen** — enter job search data day by day
3. **Summary Screen** — view weekly results and personalised feedback

## Built With
- Kotlin
- Android SDK
- ConstraintLayout
- Multi-activity navigation with Intents

## How to Run
1. Clone the repository
2. Open in Android Studio
3. Build and run on an emulator or physical device (min SDK 26)
